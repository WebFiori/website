Class Functions Details

public function function __construct(boolean $textOnly, string $text)
Constructs new list item

Parameters:

  • boolean $textOnly [Optional] [Optional] Set to TRUE to make the list item accepts text only. Default is FALSE.
  • string $text [Optional] [Optional] The text that will be displayed by the list item. Ignored if the parameter $textOnly is set to FALSE.
public function function addChild(HTMLNode $node)
Adds new child node to the list item.

Parameters:

  • HTMLNode $node The node that will be added. Node that the function will work only if the list item allows things other than the text in its body.
public function function isTextOnly()
Checks if the node only accepts text or not.
public function function setText(string $text)
Sets the text to display on the list item.

Parameters:

  • string $text The text to display. Only set if the node accepts text.

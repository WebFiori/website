File

A class that represents a file. This class can be used to read and write files in binary. In addition to that, it can be used to view files in web browsers.

Class Functions Details

public function function __construct()
Creates new instance of the class. This function will set the path and name to empty string. Also, it will set the size to 0 and ID to -1. Finally, it will set MIME type to "application/octet-stream"
public function function __toString()
Returns JSON string that represents basic file info.
public function function getAbsolutePath()
Returns the full path to the file. The full path of the file is a string that contains the path of the file alongside its name. Assuming that the path is set to "C:/Users/Me/Documents" and file name is set to "my-doc.docx", This function will return something like "C:\Users\Me\Documents\my-do.docx".
public function function getFileMIMEType()
Returns MIME type of the file. Note that if the file is specified by its path and name, the function File::read() must be called before calling this function to update its MIME type.
public function function getID()
Returns the ID of the file. This function is helpful in case the file is stored in database.
public static function function getMIMEType(string $ext)
Returns MIME type of a file type. The function will try to find MIME type based on its extension. If

Parameters:

  • string $ext File extension without the suffix (such as 'jpg').
public function function getName()
Returns the name of the file. The name is used to construct the absolute path of the file in addition to its path.
public function function getPath()
Returns the path of the file. The path is simply the folder that contains the file. For example, the path can be something like "C:\Users\Me\Documents". Note that the returned path will be using backward slashes "\".
public function function getRawData()
Returns the raw data of the file. The raw data is simply a string. It can be binary string or any basic string.
public function function getSize()
Returns the size of the file in bytes. Note that if the file is specified by its path and name, the function File::read() must be called before calling this function to update its size.
public function function read()
Reads the file in binary mode. First of all, this function checks the existence of the file. If it is exist, it tries to open the file in binary mode 'rb'. If a resource is created, it is used to read the content of the file. Also, the function will try to set MIME type of the file. If MIME type was not detected, it will set to 'application/octet-stream'. If the function is unable to read the file, it will throw an exception.
public function function remove()
Removes a file given its name and path. Before calling this function, the name of the file and its path must be specified.
public function function setID(string $id)
Sets the ID of the file. This function is helpful in case the file is stored in database.

Parameters:

  • string $id The unique ID of the file.
public function function setMIMEType(string $type)
Sets the MIME type of the file. It is not recommended to update MIME type of the file manually. Only use this function for custom file types. MIME type will be set only if its non-empty string.

Parameters:

  • string $type MIME type (such as 'application/pdf')
public function function setName(string $name)
Sets the name of the file (such as 'my-image. png') The name is used to construct the absolute path of the file in addition to its path. The name of the file must include its extension (or suffix).

Parameters:

  • string $name The name of the file.
public function function setPath(string $path)
Sets the path of the file. The path is simply the folder that contains the file. For example, the path can be something like "C:/Users/Me/Documents". The path can use forward slashes or backward slashes.

Parameters:

  • string $path The folder which will contain the file. It must be non-empty string in order to set.
public function function setRawData(string $raw)
Sets the binary representation of the file. The raw data is simply a string. It can be binary string or any basic string. Also, it can be a blob which was retrieved from a database.

Parameters:

  • string $raw Binary raw data of the file.
public function function toJSON()
Returns a JSON string that represents the file.
public function function view(boolean $asAttachment)
Display the file. If the raw data of the file is NULL, the function will try to read the file that was specified by the name and its path. If the function is unable to read the file, an exception is thrown.

Parameters:

  • boolean $asAttachment [Optional] [Optional] If this parameter is set to TRUE, the header 'content-disposition' will have the attribute 'attachment' set instead of 'inline'. This will trigger 'save as' dialog to appear.
public function function write(string $path)
Write raw binary data into a file. The function will write the data using the binary write mode 'wb' mode. This means that if the file does not exist, the function will try to created it. If it fails, It will throw an exception. Note that if an optional path is provided, the name of the file must be set first.

Parameters:

  • string $path [Optional] [Optional] An optional file path such as "C:\Users\Me\Documents". The path should not include the name of the file. If not provided, the path that is returned by File::getPath() will be used.

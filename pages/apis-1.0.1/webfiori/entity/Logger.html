Class Functions Details

public static function function callStack()
Returns a stack which contains all the called functions.
public static function function clear()
Removes the whole content of the log file. Once the content of the log is cleared, a message at the top of the log will appear. The message will say the following:

---------------Log Cleared At YYYY-MM-DD HH:MM:SS +00---------------

The '+00' is the code of the time zone.
public static function function directory(string $new)
Sets or returns the full directory of the log file. Note that If the given directory does not exists, the function will try to create it. The default place for saving logs is ROOT_DIR.'/logs'.

Parameters:

  • string $new [Optional] If provided, the save directory will be set to the given one.
public static function function displayLog()
Show log content as output on screen. This function simply open the log file and display it as output using 'echo' command.
public static function function enabled(boolean $isEnabled)
Enable, disable or check if logging is enabled.

Parameters:

  • boolean $isEnabled [Optional] If provided and set to true, logging will be enabled. If provided and not true, logging will be disabled.
public static function function get()
Returns a singleton of the class.
public static function function log(string $message, string $messageType, string $logName, boolean $addDashes)
Writes a message to the log file.

Parameters:

  • string $message The message that will be written.
  • string $messageType [Optional] [Optional] The type of the message that will be logged. it can have one of 4 values, 'info', 'warning', 'error' or 'debug'. Note that the last type will be logged only if the constant 'DEBUG' is defined. The default value is 'info'.
  • string $logName [Optional] [Optional] The name of the log file. If it is not NULL, the log will be written to the given file name.
  • boolean $addDashes [Optional] [Optional] If set to true, a line of dashes will be inserted after the message. Used to organize log messages.
public static function function logFuncCall(string $funcName, string $logFileName, string $addDashes)
Adds a debug message to a log file that says the given function was called. The message will be logged only if the constant 'DEBUG' is defined.

Parameters:

  • string $funcName The name of the function. To get the name of the function in its body, Use the magic constant '__METHOD__' or '__FUNCTION__'. It is recommended to always use '__METHOD__' as this constant will return class name with it if the function is inside a class.
  • string $logFileName [Optional] [Optional] The name of the log file. If it is not NULL, the log will be written to the given file name.
  • string $addDashes [Optional] [Optional] If set to true, a line of dashes will be inserted after the message. Used to organize log messages.
public static function function logFuncReturn(string $funcName, string $logFileName, string $addDashes)
Adds a debug message to a log file that says the execution of a given function was finished. Note that the message will be logged only if the constant 'DEBUG' is defined. To get the name of the function in its body, Use the magic constant '__METHOD__' or '__FUNCTION__'. It is recommended to always use '__METHOD__' as this constant will return class name with it if the function is inside a class.

Parameters:

  • string $funcName The name of the function.
  • string $logFileName [Optional] The name of the log file. If it is not NULL, the log will be written to the given file name. Default is NULL.
  • string $addDashes [Optional] If set to true, a line of dashes will be inserted after the message. Used to organize log messages.
public static function function logName(string $new)
Sets or returns the name of the log file. This function is used to switch between different log files. The name should be provided without any extentions (e.g. 'my-log'). Note that log files will always have the extention .txt The default log file name is 'log.txt'.

Parameters:

  • string $new [Optional] The name of the log file that the system will be writing logs to.
public static function function logReturnValue(mixed $val, type $logName, boolean $addDashes)
Adds a log message to log function return value (debug).

Parameters:

  • mixed $val The return value of a function.
  • type $logName [Optional] [Optional] The name of the log file. If it is not NULL, the log will be written to the given file name.
  • boolean $addDashes [Optional] If set to true, a line of dashes will be inserted after the message. Used to organize log messages.
public static function function requestCompleted()
Adds a message to the last selected log file that states the client request was processed. This function is usually called after calling the function 'die()' or 'exit()'. Also if no server code will be executed after. The exact message that will be logged is:

"Processing of client request is finished."

public static function function section()
Adds a new line to separate log parts. The line will have the following text:

-+--+-

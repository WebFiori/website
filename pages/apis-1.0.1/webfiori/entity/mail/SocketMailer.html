public function function addReceiver(string $name, string $address, boolean $isCC, boolean $isBcc)
Adds new receiver.

Parameters:

  • string $name The name of the email receiver (such as 'Ibrahim').
  • string $address The email address of the receiver.
  • boolean $isCC [Optional] If set to true, the receiver will receive a carbon copy of the message.
  • boolean $isBcc [Optional] If set to true, the receiver will receive a blind carbon copy of the message.

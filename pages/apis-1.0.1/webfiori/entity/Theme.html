Class Functions Details

public function function __construct()
Creates new instance of the class using default values. The default values will be set as follows:
  • Theme name will be an empty string.
  • Theme URL will be an empty string.
  • Author name will be an empty string.
  • Author URL will be an empty string.
  • Theme version will be set to '1.0.0'
  • Theme license will be an empty string.
  • License URL will be an empty string.
  • Theme description will be an empty string.
  • Theme directory name will be an empty string.
  • Theme CSS directory name will be set to 'css'
  • Theme JS directory name will be set to 'js'
  • Theme images directory name will be set to 'images'
public function function addComponent(string $componentName)
Adds a single component to the set of theme components. Theme components are a set of PHP files that must exist inside theme directory.

Parameters:

  • string $componentName The name of the component file (such as 'head.php')
public function function addComponents(array $arr)
Adds a set of theme components to the theme. Theme components are a set of PHP files that must exist inside theme directory. The developer can create any number of components and add them to the theme.

Parameters:

  • array $arr An array that contains the names of components files (such as 'head.php').
public function function getAuthor()
Returns the name of theme author.
public function function getAuthorUrl()
Returns the URL which takes the users to author's web site.
public static function function getAvailableThemes()
Returns an array that contains the meta data of all available themes. This function will return an associative array. The key is the theme name and the value is an object of type Theme that contains theme info.
public function function getComponents()
Returns an array which contains the names of theme components files. Theme components are a set of PHP files that must exist inside theme directory.
public function function getCssDirName()
Returns the name of the directory where CSS files are kept.
public function function getDescription()
Returns the description of the theme.
public function function getDirectoryName()
Returns the name of the directory where all theme files are kept.

Returns the name of the directory where theme images are kept.
public function function getJsDirName()
Returns the name of the directory where JavaScript files are kept.
public function function getLicenseName()
Returns the name of theme license.
public function function getLicenseUrl()
Returns a URL which should contain a full version of theme license.
public static function function getLoadedThemes()
Returns an array which contains all loaded themes.
public function function getName()
Returns the name of the theme. If the name is not set, the function will return empty string.
public function function getUrl()
Returns A URL which should point to theme web site.
public function function getVersion()
Returns theme version number.
public function function invokeAfterLoaded()
Fire the callback function. This function must not be used by the developers. It is called automatically when the theme is loaded.
public function function invokeBeforeLoaded()
Fire the callback function. This function must not be used by the developers. It is called automatically when the theme is being loaded.
public static function function isThemeLoaded(string $themeName)
Checks if a theme is loaded or not given its name.

Parameters:

  • string $themeName The name of the theme.
public function function setAfterLoaded(callable $function, array $params)
Sets the value of the callback which will be called after theme is loaded.

Parameters:

  • callable $function The callback.
  • array $params [Optional] An array of parameters which can be passed to the callback.
public function function setAuthor(string $author)
Sets the name of theme author.

Parameters:

  • string $author The name of theme author (such as 'Ibrahim BinAlshikh')
public function function setAuthorUrl(string $authorUrl)
Sets the URL to the theme author. It can be the same as Theme URL.

Parameters:

  • string $authorUrl The URL to the author's web site.
public function function setBeforeLoaded(callback $function, array $params)
Sets the value of the callback which will be called before theme is loaded.

Parameters:

  • callback $function The callback.
  • array $params [Optional] An array of parameters which can be passed to the callback.
public function function setCssDirName(string $name)
Sets the name of the directory where theme CSS files are kept.

Parameters:

  • string $name The name of the directory where theme CSS files are kept. Note that it will be set only if the given name is not an empty string.
public function function setDescription(string $desc)
Sets the description of the theme.

Parameters:

  • string $desc Theme description. Usually a short paragraph of two or 3 sentences.
public function function setDirectoryName(string $name)
Sets the name of the directory where all theme files are kept. Each theme must have a unique directory to prevent collision.

Parameters:

  • string $name The name of the directory.
public function function setImagesDirName(string $name)
Sets the name of the directory where theme images are kept.

Parameters:

  • string $name The name of the directory where theme images are kept. Note that it will be set only if the given name is not an empty string.
public function function setJsDirName(string $name)
Sets the name of the directory where theme JavaScript files are kept.

Parameters:

  • string $name The name of the directory where theme JavaScript files are kept. Note that it will be set only if the given name is not an empty string.
public function function setLicenseName(string $text)
Sets the name of theme license.

Parameters:

  • string $text The name of theme license.
public function function setLicenseUrl(string $url)
Sets a URL to the license where people can find more details about it.

Parameters:

  • string $url A URL to the license.
public function function setName(string $name)
Sets the name of the theme.

Parameters:

  • string $name The name of the theme. Note that the name of the theme acts as the unique identifier for the theme.
public function function setUrl(string $url)
Sets the URL of theme designer web site. Theme URL can be the same as author URL.

Parameters:

  • string $url The URL to theme designer web site.
public function function setVersion(string $vNum)
Sets the version number of the theme.

Parameters:

  • string $vNum Version number. The format of version number is usually like 'X.X.X' where the 'X' can be any number.
public function function toJSON()
Returns an object of type JsonX that represents the theme. JSON string that will be generated by the JsonX instance will have the following information:

{
  "name":""
  "version":""
  "author":""
  "images-dir-name":""
  "theme-dir-name":""
  "css-dir-name":""
  "js-dir-name":""
  "components":[]
}

public static function function usingTheme(string $themeName)
Loads a theme given its name. If the given name is NULL, the function will load the default theme as specified by the function SiteConfig::getBaseThemeName().

Parameters:

  • string $themeName [Optional] The name of the theme.

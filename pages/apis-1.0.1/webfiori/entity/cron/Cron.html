namespace \webfiori\entity\cron

Cron

A class that is used to manage cron jobs. It is used to create jobs, schedule them and execute them. In order to run the jobs automatically, the developer must add an entry in the following formate in crontab:

/usr/bin/curl {BASE_URL}/cron-jobs/execute/{password}

Where {BASE_URL} is the web site's base URL and {password} is the password that was set by the developer to protect the jobs from unauthorized access.

Class Functions Details

public static function function createJob(string $when, callable $function, array $funcParams, string $jobName)
Creates new job using cron expression. The job will be created and scheduled only if the given cron expression is valid. For more information on cron expressions, go to https://en.wikipedia.org/wiki/Cron#CRON_expression. Note that the function does not support year field. This means the expression will have only 5 fields.

Parameters:

  • string $when [Optional] A cron expression.
  • callable $function [Optional] A function to run when it is the time to execute the job.
  • array $funcParams [Optional] An array of parameters that can be passed to the function.
  • string $jobName An optional job name.
public static function function dailyJob(string $time, callable $func, array $funcParams)
Creates a daily job to execute every day at specific hour and minute.

Parameters:

  • string $time A time in the form 'hh:mm'. hh can have any value between 0 and 23 inclusive. mm can have any value between 0 and 59 inclusive.
  • callable $func A function that will be executed once it is the time to run the job.
  • array $funcParams [Optional] An array of parameters which will be passed to the function.
public static function function execLog(boolean $bool)
Enable or disable logging for jobs execution. This function is also used to check if logging is enabled or not.

Parameters:

  • boolean $bool [Optional] [Optional] If set to TRUE, a log file that contains the details of the executed jobs will be created in 'logs' folder. Default value is NULL.
public static function function jobsQueue()
Returns a queue of all queued jobs.
public static function function password(string $pass)
Sets or gets the password that is used to protect the cron instance. The password is used to prevent unauthorized access to execute jobs.

Parameters:

  • string $pass [Optional] If not NULL, the password will be updated to the given one.
public static function function scheduleJob(CronJob $job)
Adds new job to jobs queue.

Parameters:

  • CronJob $job An instance of the class 'CronJob'.
public static function function weeklyJob(string $time, function $func, array $funcParams)
Creates a job that will be executed on specific time weekly.

Parameters:

  • string $time A string in the format 'd-hh:mm'. 'd' can be a number between 0 and 6 inclusive or a 3 characters day name such as 'sun'. 0 is for Sunday and 6 is for Saturday. 'hh' can have any value between 0 and 23 inclusive. mm can have any value between 0 and 59 inclusive.
  • function $func A function that will be executed once it is the time to run the job.
  • array $funcParams [Optional] An array of parameters which will be passed to the function.

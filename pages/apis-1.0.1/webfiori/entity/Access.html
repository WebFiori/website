Class Functions Details

public static function function &getGroup(string $groupId)
Returns an object of type UsersGroup given its ID. This function can be used to check if a group is exist or not. If the function has returned NULL, this means the group does not exist.

Parameters:

  • string $groupId The ID of the group.
public static function function &getPrivilege(string $id)
Returns a privilege object given privilege ID. This function will search all created groups for a privilege which has the given ID. If not found, the function will return NULL. This function also can be used to check if a privilege is exist or not. If the function has returned NULL, this means the privilege does not exist.

Parameters:

  • string $id The ID of the privilege.
public static function function cloneGroup(string $groupId, type $newGroupId, type $newPermissions)
Creates a clone of a group given its ID. The clone group will contain all the privileges which where added to the original group. Also, it is possible to add extra privileges by suppling an array that contains the IDs of the new privileges.

Parameters:

  • string $groupId The ID of the group that will be cloned.
  • type $newGroupId The ID of the new group. It must be unique.
  • type $newPermissions [Optional] An optional array of new privileges array.
public static function function createPermissionsStr(User $user)
Creates a string of permissions given a user. This function can be handy in case the developer would like to store user privileges in a database. The function might return a string which might looks like the following string:

'PRIVILEGE_1-1;PRIVILEGE_2-1;G-A_GROUP'

where 'PRIVILEGE_1' and 'PRIVILEGE_2' are IDs of privileges and 'A_GROUP' is the ID of a group that the user belongs to. The number that comes after the dash is the status of the privilege. Each privilege or a group will be separated from the other by a semicolon. Also the group will have the letter 'G' at the start.

Parameters:

  • User $user The user which the permissions string will be created from.
public static function function groups()
Returns an array which contains all user groups. The array will be empty if no user groups has been created.
public static function function hasGroup(string $groupId)
Checks if a users group does exist or not given its ID.

Parameters:

  • string $groupId The ID of the group.
public static function function hasPrivilege(string $id)
Checks if a privilege does exist or not given its ID. The function will search all created groups for a privilege with the given ID.

Parameters:

  • string $id The ID of the privilege.
public static function function newGroup(string $groupId)
Creates new users group using specific ID. The group is the base for user privileges. After creating it, the developer can add a set of privileges to the group. Note that the group will not created only if the name of the group contains invalid characters or it is already created.

Parameters:

  • string $groupId The ID of the group. The ID must not contain any of the following characters: ';','-' or a space. If the name contains any of the given characters, the group will not created.
public static function function newPrivilege(string $groupId, string $privilegeId)
Creates new privilege in a specific group given its ID.

Parameters:

  • string $groupId The ID of the group that the privilege will be added to. It must be a group in the groups array of the access class.
  • string $privilegeId The ID of the privilege. The ID must not contain any of the following characters, ';','-' or a space.
public static function function newPrivileges(string $groupId, array $prNamesArr)
Creates multiple privileges in a group given its ID. This function can be used as a shorthand to create multiple privileges in a group instead of calling Access::newPrivilege() multiple times.

Parameters:

  • string $groupId The ID of the group. The group must be created before starting to create privileges in it.
  • array $prNamesArr An associative array that contains the names of privileges.
public static function function privileges(string|NULL $groupId)
Returns an array which contains all privileges or privileges in a specific user group.

Parameters:

  • string|NULL $groupId [Optional] [Optional] The ID of the group which its privileges will be returned. If NULL is given, all privileges will be returned. Default is NULL.
public static function function resolvePriviliges(string $str, User $user)
Adds privileges to a user given privileges string.

Parameters:

  • string $str A string of privileges. The format of the string must follow the following format: 'PRIVILEGE_1-0;PRIVILEGE_2-1;G-A_GROUP' where 'PRIVILEGE_1' and 'PRIVILEGE_2' are IDs of privileges and 'A_GROUP' is the ID of a group that the user belongs to. The number that comes after the dash is the status of the privilege. If 0, then the user will not have the given privilege. If 1, the user will have the privilege. In the given example, The user will have only 'PRIVILEGE_2'. and he will belong to the group that has the ID 'A_GROUP'. Each privilege or a group must be separated from the other by a semicolon. Also the group must have the letter 'G' at the start. Note that in the given example, if 'PRIVILEGE_1' is in 'A_GROUP', he will not have it even if it is in group permissions.
  • User $user The user which the permissions will be added to

public function function useDatabase(array $optionalConnectionParams)
Initiate database connection.

Parameters:

  • array $optionalConnectionParams [Optional] [Optional] An associative array which contains database connection information. The indices of the array are:
    • host: Database host address.
    • port: Port number.
    • user: Database username.
    • pass: Database user's password.
    • db-name: The name of the database.
    If this parameter is not provided, connection information will be taken from the class 'Config'.

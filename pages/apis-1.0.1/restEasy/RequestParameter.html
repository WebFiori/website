Parameters:

  • string $name The name of the parameter as it appears in the request body. It must be a valid name> If the given name is invalid, the parameter name will be set to 'a-parameter'.
  • string $type [Optional] The type of the data that will be in the parameter (integer, string, email etc...). It must be a value from the array APIFilter::TYPES. If the given type is invalid, 'string' is used.
  • boolean $isOptional [Optional] Set to TRUE if the parameter is optional.

public function function setCustomFilterFunction(callback $function, boolean $applyBasicFilter)
Sets a callback function to work as a filter for request parameter. The callback function will have two parameters passed to it. The first one is an associative array that contains the not-filtered value and the value filtered using basic filter. The values are contained in two indices:
  • original-value
  • basic-filter-result
If the parameter $applyBasicFilter is set to FALSE, the index 'basic-filter-result' will have the value 'NOT_APLICABLE'. The second parameter is an object of type RequestParameter which contains original information for the filter. The function must be implemented in a way that makes it return FALSE if the parameter has invalid value. If the parameter is filtered and was validated, the function must return the valid and filtered value.

Parameters:

  • callback $function A callback function.
  • boolean $applyBasicFilter [Optional] If set to TRUE, the basic filter will be applied to the parameter. Default is TRUE.

public function function setMaxVal(int $val)
Sets the maximum value. The value will be updated only if:
  • The request parameter type is numeric ('integer').
  • The given value is greater than RequestParameter::getMinVal()

Parameters:

  • int $val The maximum value to set.
public function function setMinVal(int $val)
Sets the minimum value that the parameter can accept. The value will be updated only if:
  • The request parameter type is numeric ('integer').
  • The given value is less than RequestParameter::getMaxVal()

Parameters:

  • int $val The minimum value to set.
public function function setName(string $name)
Sets the name of the parameter. A valid parameter name must follow the following rules:
  • It can contain the letters [A-Z] and [a-z].
  • It can contain the numbers [0-9].
  • It can have the character '-' and the character '_'.

Parameters:

  • string $name The name of the parameter.
public function function setType(string $type)
Sets the type of the parameter.

Parameters:

  • string $type The type of the parameter. It must be a value form the array APIFilter::TYPES.
public function function toJSON()
Returns a JsonX object that represents the request parameter. This function is used to help front-end developers in showing the documentation of the request parameter.

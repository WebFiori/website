public function function select(array $selectOptions)
Constructs a 'select' query.

Parameters:

  • array $selectOptions [Optional] An associative array which contains options to construct different select queries. The available options are:
    • colums: An optional array which can have the keys of columns that will be select.
    • limit: The 'limit' attribute of the query.
    • offset: The 'offset' attribute of the query. Ignored if the option 'limit' is not set.
    • condition-cols-and-vals: An associative array. The indices can be values the value at each index is an objects of type 'Column'. Or the indices can be column indices taken from MySQLTable object and the values are set for each index. The second way is recommended as one table might have two columns with the same values. will be selected based on.
    • conditions: An array that can contains two possible values: '=' or '!='. If anything else is given at specific index, '=' will be used.
    • join-operators: An array that contains a set of MySQL join operators like 'and' and 'or'.
    • select-max: A boolean value. Set to TRUE if you want to select maximum value of a column. Ignored in case the option 'columns' is set.
    • select-min: A boolean value. Set to TRUE if you want to select minimum value of a column. Ignored in case the option 'columns' or 'select-max' is set.
    • column: The column which contains maximum or minimum value.
    • rename-to: Rename the max or min column to the given name.
    • order-by: An object of type column at which the rows will be ordered by.
    • order-type: A one character string. 'A' for ascending and 'D' for descending. Default is 'A'. Used only if 'order-by' is set.

public function function selectByColVal(string $col, string $val, string $cond, int $limit, int $offset)
Constructs a query that can be used to get table data based on a specific column value.

Parameters:

  • string $col The name of the column in the table.
  • string $val The value that is used to filter data.
  • string $cond [Optional] The condition of select statement. It can be '=' or '!='. If anything else is given, '=' will be used. Note that if the parameter $val is equal to 'IS NULL' or 'IS NOT NULL', This parameter is ignored. Default is '='.
  • int $limit [Optional] The value of the attribute 'limit' of the select statement. If zero or a negative value is given, it will not be included in the generated MySQL query. Default is -1.
  • int $offset [Optional] [Optional] The value of the attribute 'offset' of the select statement. If zero or a negative value is given, it will not be included in the generated MySQL query. Default is -1.

public function function setMaxPackete(int $size, string $unit)
Constructs a query which can be used to update the server's global variable 'max_allowed_packet'. The value of the attribute is in bytes. The developer might want to update the value of this variable if he wants to send large data to database using one query. The maximum value this attribute can have is 1073741824 bytes.

Parameters:

  • int $size The new size.
  • string $unit [Optional] One of 4 values: 'B' for byte, 'KB' for kilobyte, 'MB' for megabyte and 'GB' for gigabyte. If the given value is none of the 4, the type will be set to 'MP'.

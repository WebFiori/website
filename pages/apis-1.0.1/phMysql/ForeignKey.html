public function function __construct(string $name, string $sourceTblName, array|string $sourceCol, string $refTblName, string $refCol)
Creates new foreign key.

Parameters:

  • string $name The name of the key. It must be a string and its not empty. Also it must not contain any spaces or any characters other than A-Z, a-z and underscore. The default value is 'key_name'.
  • string $sourceTblName The name of the table that will contain the key. It must be a string and its not empty. Also it must not contain any spaces or any characters other than A-Z, a-z and underscore. The default value is 'NULL'.
  • array|string $sourceCol A name or an array of names that contain source columns names. The name or names must not contain any spaces or any characters other than A-Z, a-z and underscore. The default value is 'NULL'.
  • string $refTblName The name of the referenced table. It must be a string and its not empty. Also it must not contain any spaces or any characters other than A-Z, a-z and underscore. The default value is 'NULL'.
  • string $refCol A name or an array of names that contain referenced columns names. The name or names must not contain any spaces or any characters other than A-Z, a-z and underscore. The default value is 'NULL'.

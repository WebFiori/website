public function function __construct(string $tName)
Creates a new instance of the class. This function will initialize the basic settings of the table. It will set MySQL version to 5.5, the engine to 'InnoDB', char set to 'utf8mb4' and the order to 0.

Parameters:

  • string $tName The name of the table. It must be a string and its not empty. Also it must not contain any spaces or any characters other than A-Z, a-z and underscore. If the given name is invalid or not provided, 'table' will be used as default.

Adds new column to the table.

Parameters:

  • Column $col An object of type Column. Note that the column will be added only if no column was found in the table which has the same name as the given column.

  • ForeignKey $key an object of type 'ForeignKey'.
public function function addMultiReference(MySQLTable $refTable, array $refColsArr, array $targetColsArr, string $keyname, string $onupdate, string $ondelete)
Adds a foreign key which references multiple columns.

Parameters:

  • MySQLTable $refTable The referenced table.
  • array $refColsArr An array which contains the names of referenced columns. The names of columns must in the referenced table. If one of the names is passed to the function MySQLTable::getColumn(), it should return an object of type 'Column'.
  • array $targetColsArr An array which contains the targeted columns.
  • string $keyname The name of the key.
  • string $onupdate [Optional] The 'on update' condition for the key. it can be one of the following:
    • set null
    • cascade
    • restrict
    • set default
    • no action
    Default value is 'set null'.
  • string $ondelete [Optional] The 'on delete' condition for the key. it can be one of the following:
    • set null
    • cascade
    • restrict
    • set default
    • no action
    Default value is 'set null'.
public function function addReference(MySQLTable $refTable, string $refColName, string $targetCol, string $keyname, string $onupdate, string $ondelete)
Adds a foreign key to the table.

Parameters:

  • MySQLTable $refTable The table that will be referenced.
  • string $refColName The name of the column that will be referenced. It must be a column in the referenced table. The value of this attribute is a value that once passed to the function MySQLTable::getColumn() will return an object of type 'Column'.
  • string $targetCol The target column. It must be a column in the current instance. The value of this attribute is a value that once passed to the function MySQLTable::getColumn() will return an object of type 'Column'.
  • string $keyname The name of the foreign key.
  • string $onupdate [Optional] The 'on update' condition for the key. it can be one of the following:
    • set null
    • cascade
    • restrict
    • set default
    • no action
    Default value is 'set null'.
  • string $ondelete [Optional] The 'on delete' condition for the key. it can be one of the following:
    • set null
    • cascade
    • restrict
    • set default
    • no action
    Default value is 'set null'.

public function function setMySQLVersion(string $vNum)
Sets version number of MySQL server. Version number of MySQL is used to set the correct collation for table columns in case of varchar or text data types. If MySQL version is '5.5' or lower, collation will be set to 'utf8mb4_unicode_ci'. Other than that, the collation will be set to 'utf8mb4_unicode_520_ci'.

Parameters:

  • string $vNum MySQL version number (such as '5.5').
public function function setName(string $param)

Parameters:

  • string $param The name of the table (such as 'users'). It must be a string and its not empty. Also it must not contain any spaces or any characters other than A-Z, a-z and underscore.
public function function setOrder(int $val)
Sets the order of the table in the database. The order of the table describes the dependencies between tables. For example, if we have three tables, 'A', 'B' and 'C'. Let's assume that table 'B' references table 'A' and Table 'A' references table 'C'. In this case, table 'C' will have order 0, Table 'A' have order 1 and table 'B' have order 2.

Parameters:

  • int $val The order of the table in the database.
